# Cluster Analysis Learning Project

This repository contains my learning materials, notebooks, scripts, and experiments related to cluster analysis and unsupervised machine learning.

## Topics

- Data preprocessing
- Exploratory data analysis
- Distance metrics
- K-means clustering
- Hierarchical clustering
- DBSCAN
- Cluster visualization
- Evaluation of clustering results

## Structure

- `notebooks/` — Jupyter notebooks
- `data/` — datasets used for practice
- `scripts/` — reusable Python scripts
- `environment.yml` — conda environment definition
